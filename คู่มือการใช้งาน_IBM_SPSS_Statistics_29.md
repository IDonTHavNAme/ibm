## 1. IBM SPSS Statistics คืออะไร?

**IBM SPSS Statistics** (Statistical Package for the Social Sciences) คือโปรแกรมวิเคราะห์ข้อมูลทางสถิติที่ IBM พัฒนาขึ้น ใช้งานมากกว่า 50 ปีในแวดวงวิชาการ งานวิจัย และธุรกิจ

### จุดเด่นของ SPSS
| คุณสมบัติ | รายละเอียด |
|---|---|
| ใช้งานง่าย | มีเมนูแบบ Point-and-Click ไม่ต้องเขียนโค้ด |
| ครบถ้วน | รองรับการทดสอบทางสถิติมากกว่า 100 แบบ |
| แม่นยำ | ใช้อ้างอิงในงานวิจัยระดับนานาชาติได้ |
| Output ชัดเจน | ผลลัพธ์แสดงเป็นตาราง พร้อมค่า p-value และ sig. |
| รองรับไฟล์หลายประเภท | Excel, CSV, SAS, Stata, Text |

### SPSS ต่างจากโปรแกรมอื่นอย่างไร?
```
Excel       → เหมาะกับการคำนวณทั่วไป ไม่ใช่สถิติเชิงลึก
Python/R    → ยืดหยุ่นกว่า แต่ต้องเขียนโค้ด
SPSS        → สมดุลระหว่าง "ง่าย" และ "ครบถ้วน" ที่สุดสำหรับนักวิจัย
```

## 2. ใครควรใช้โปรแกรมนี้?

- **นักศึกษาระดับปริญญาตรี-โท-เอก** — วิเคราะห์ข้อมูลวิทยานิพนธ์
- **นักวิจัย** — ทดสอบสมมติฐาน, วิเคราะห์ผลแบบสอบถาม
- **นักการตลาด** — วิเคราะห์พฤติกรรมผู้บริโภค
- **ฝ่าย HR** — วิเคราะห์ข้อมูลพนักงาน, ความพึงพอใจ
- **บุคลากรสาธารณสุข** — วิจัยทางคลินิก, Epidemiology
- **หน่วยงานรัฐ** — สำรวจประชากร, นโยบายสาธารณะ

## 3. ทำความรู้จักหน้าตาโปรแกรม

เมื่อเปิดโปรแกรมขึ้นมา จะเห็นหน้าต่างหลัก **Data Editor** แบ่งเป็น 2 แท็บ:

```
┌─────────────────────────────────────────────────────┐
│  IBM SPSS Statistics Data Editor                    │
├─────────────────────────────────────────────────────┤
│  File | Edit | View | Data | Transform | Analyze   │
│  Graphs | Utilities | Extensions | Window | Help   │
├─────────────────────────────────────────────────────┤
│                                                     │
│   [พื้นที่แสดงข้อมูล — ตารางแถว x คอลัมน์]         │
│                                                     │
├─────────────────────────────────────────────────────┤
│  [Data View]  [Variable View]                       │
└─────────────────────────────────────────────────────┘
```

### แท็บ Data View
> คือ **ตารางข้อมูล** เหมือน Excel
> - **แถว (Row)** = แต่ละกรณี/ผู้ตอบแบบสอบถาม (Case)
> - **คอลัมน์ (Column)** = แต่ละตัวแปร (Variable)

### แท็บ Variable View
> คือ **การตั้งค่าตัวแปร** เช่น ชื่อ, ประเภทข้อมูล, Label
> ต้องตั้งค่าตรงนี้ก่อนวิเคราะห์ข้อมูลทุกครั้ง

### หน้าต่าง Output Viewer
> เมื่อรันการวิเคราะห์ ผลลัพธ์จะแสดงที่หน้าต่างแยกต่างหาก
> ประกอบด้วยตาราง (Pivot Table) และกราฟ

### เมนูหลักที่ใช้บ่อย
| เมนู | ใช้ทำอะไร |
|---|---|
| `File` | เปิด/บันทึก/นำเข้าข้อมูล |
| `Data` | Sort, Filter, Merge ข้อมูล |
| `Transform` | Compute Variable, Recode |
| `Analyze` | รันการวิเคราะห์สถิติทุกประเภท |
| `Graphs` | สร้างกราฟ |

## 4. เริ่มต้นใช้งาน Step by Step

### Step 1 — เปิดโปรแกรม
1. ดับเบิลคลิก Icon SPSS Statistics 29 บน Desktop
2. โปรแกรมจะแสดง Welcome Dialog
3. เลือก **"New Dataset"** เพื่อเริ่มต้นใหม่ หรือ **"Open an existing file"** เพื่อเปิดไฟล์เดิม

### Step 2 — สร้างตัวแปรใน Variable View
1. คลิกแท็บ **Variable View** (ด้านล่างซ้าย)
2. พิมพ์ชื่อตัวแปรในคอลัมน์ **Name** เช่น `age`, `gender`, `score`
3. กำหนด **Type**: Numeric (ตัวเลข) หรือ String (ตัวอักษร)
4. กำหนด **Label**: ชื่อเต็มของตัวแปร เช่น "อายุ (ปี)"
5. กำหนด **Values**: ถ้าเป็นข้อมูล Nominal เช่น 1=ชาย, 2=หญิง
6. กำหนด **Measure**: Scale / Ordinal / Nominal

```
ตัวอย่างการตั้งค่า:
┌────────┬─────────┬───────┬────────────┬──────────┬─────────┐
│ Name   │ Type    │ Width │ Label      │ Values   │ Measure │
├────────┼─────────┼───────┼────────────┼──────────┼─────────┤
│ id     │ Numeric │  10   │ รหัสผู้ตอบ │ None     │ Scale   │
│ gender │ Numeric │   8   │ เพศ        │ 1=ชาย    │ Nominal │
│        │         │       │            │ 2=หญิง   │         │
│ age    │ Numeric │   8   │ อายุ       │ None     │ Scale   │
│ score  │ Numeric │   8   │ คะแนน      │ None     │ Scale   │
└────────┴─────────┴───────┴────────────┴──────────┴─────────┘
```

### Step 3 — กรอกข้อมูลใน Data View
1. คลิกแท็บ **Data View**
2. กรอกข้อมูลในแต่ละช่อง (เหมือน Excel)
3. กด Enter เพื่อเลื่อนไปแถวถัดไป, Tab เพื่อเลื่อนคอลัมน์

### Step 4 — วิเคราะห์ข้อมูล
1. คลิกเมนู **Analyze**
2. เลือกประเภทการวิเคราะห์ที่ต้องการ
3. ลากตัวแปรที่ต้องการเข้าช่อง
4. คลิก **OK**
5. ผลลัพธ์จะปรากฏใน Output Viewer

### Step 5 — บันทึกไฟล์
- **บันทึกข้อมูล**: `File → Save As` → เลือก `.sav` (ไฟล์ข้อมูล SPSS)
- **บันทึก Output**: ใน Output Viewer → `File → Save As` → `.spv`

## 5. การนำเข้าข้อมูล (Import Data)

### นำเข้าจาก Excel (.xlsx)
1. `File → Import Data → Excel`
2. Browse เลือกไฟล์
3. เลือก Sheet ที่ต้องการ
4. ติ๊ก **"Read variable names from the first row of data"** ถ้า Row แรกเป็นหัวคอลัมน์
5. คลิก **OK**

### นำเข้าจาก CSV (.csv)
1. `File → Import Data → CSV Data`
2. Browse เลือกไฟล์
3. ตรวจสอบ Delimiter (จุลภาค `,` หรือ Tab)
4. คลิก **OK**

### นำเข้าจากข้อมูลพิมพ์เอง
กรอกข้อมูลตรงใน Data View ได้เลย ไม่ต้องนำเข้า

> **หมายเหตุ**: ไฟล์ข้อมูลของ SPSS มีนามสกุล `.sav`
> ถ้าเปิดไฟล์ `.sav` ที่มีอยู่แล้ว: `File → Open → Data`

## 6. การกำหนดตัวแปร (Variable View)

### ประเภทของ Measure (ระดับการวัด)
| Measure | ความหมาย | ตัวอย่าง |
|---|---|---|
| **Scale** | ข้อมูลต่อเนื่อง มีหน่วย | อายุ, น้ำหนัก, คะแนน |
| **Ordinal** | ข้อมูลจัดอันดับ แต่ห่างไม่เท่ากัน | ระดับความพึงพอใจ (1-5) |
| **Nominal** | ข้อมูลจัดกลุ่ม ไม่มีลำดับ | เพศ, จังหวัด, อาชีพ |

### การกำหนด Values Labels
1. คลิกช่อง Values ของตัวแปรที่ต้องการ
2. คลิกปุ่ม `...` (สามจุด)
3. พิมพ์ **Value** (ตัวเลข) และ **Label** (ความหมาย)
4. คลิก **Add** แล้วทำซ้ำสำหรับทุกค่า
5. คลิก **OK**

```
ตัวอย่าง:
Value: 1 → Label: "น้อยที่สุด"
Value: 2 → Label: "น้อย"
Value: 3 → Label: "ปานกลาง"
Value: 4 → Label: "มาก"
Value: 5 → Label: "มากที่สุด"
```

### Missing Values (ค่าขาดหาย)
หากข้อมูลบางช่องไม่มีการตอบ:
1. คลิกช่อง Missing ของตัวแปร
2. เลือก **Discrete missing values**
3. กำหนดเลขที่ใช้แทนค่าขาดหาย เช่น `99` หรือ `-1`
4. คลิก **OK**

## 7. การวิเคราะห์ข้อมูลเบื้องต้น

### 7.1 Descriptive Statistics (สถิติเชิงพรรณนา)

ใช้เมื่อต้องการดู: ค่าเฉลี่ย, ส่วนเบี่ยงเบนมาตรฐาน, ค่าต่ำสุด/สูงสุด

**วิธีทำ:**
```
Analyze → Descriptive Statistics → Descriptives
→ ลากตัวแปรที่ต้องการเข้าช่อง Variable(s)
→ คลิก Options เลือก Mean, Std. Deviation, Min, Max
→ คลิก OK
```

**ผลลัพธ์ที่ได้ (ตัวอย่าง):**
```
Descriptive Statistics
┌──────────┬───┬─────┬──────┬────────┬───────────────────┐
│          │ N │ Min │ Max  │  Mean  │ Std. Deviation    │
├──────────┼───┼─────┼──────┼────────┼───────────────────┤
│ age      │100│  18 │  65  │ 35.20  │      12.450       │
│ score    │100│  50 │  100 │ 74.80  │       8.320       │
└──────────┴───┴─────┴──────┴────────┴───────────────────┘
```

---

### 7.2 Frequencies (ความถี่)

ใช้เมื่อต้องการนับจำนวน/เปอร์เซ็นต์ของข้อมูลกลุ่ม เช่น จำนวนชาย-หญิง

**วิธีทำ:**
```
Analyze → Descriptive Statistics → Frequencies
→ ลากตัวแปรเข้าช่อง Variable(s)
→ คลิก Statistics (ถ้าต้องการ Mean, Median)
→ คลิก OK
```

---

### 7.3 Crosstabulation (ตารางไขว้)

ใช้เมื่อต้องการดูความสัมพันธ์ระหว่างตัวแปรกลุ่ม 2 ตัว

**วิธีทำ:**
```
Analyze → Descriptive Statistics → Crosstabs
→ Row(s): ตัวแปรที่ 1
→ Column(s): ตัวแปรที่ 2
→ คลิก Statistics → ติ๊ก Chi-square
→ คลิก Cells → ติ๊ก Row, Column, Total percentages
→ คลิก OK
```

---

### 7.4 Independent Samples T-Test

ใช้เมื่อต้องการเปรียบเทียบค่าเฉลี่ยระหว่าง 2 กลุ่ม เช่น คะแนนเฉลี่ยชาย vs หญิง

**วิธีทำ:**
```
Analyze → Compare Means → Independent-Samples T Test
→ Test Variable(s): ตัวแปรตัวเลข (เช่น score)
→ Grouping Variable: ตัวแปรกลุ่ม (เช่น gender)
→ คลิก Define Groups → Group 1: 1, Group 2: 2
→ คลิก Continue → OK
```

**อ่านผลลัพธ์:**
- ดูที่ **Sig. (2-tailed)**
  - ถ้า Sig. < 0.05 → มีความแตกต่างอย่างมีนัยสำคัญทางสถิติ
  - ถ้า Sig. ≥ 0.05 → ไม่มีความแตกต่างอย่างมีนัยสำคัญ

---

### 7.5 One-Way ANOVA

ใช้เมื่อต้องการเปรียบเทียบค่าเฉลี่ยระหว่างมากกว่า 2 กลุ่ม

**วิธีทำ:**
```
Analyze → Compare Means → One-Way ANOVA
→ Dependent List: ตัวแปรผลลัพธ์ (เช่น score)
→ Factor: ตัวแปรกลุ่ม (เช่น department)
→ คลิก Post Hoc → ติ๊ก Tukey (เพื่อดูว่ากลุ่มไหนต่างกัน)
→ คลิก OK
```

---

### 7.6 Pearson Correlation (สหสัมพันธ์)

ใช้เมื่อต้องการดูความสัมพันธ์ระหว่างตัวแปรตัวเลข 2 ตัว

**วิธีทำ:**
```
Analyze → Correlate → Bivariate
→ ลากทั้ง 2 ตัวแปรเข้าช่อง Variables
→ ติ๊ก Pearson
→ คลิก OK
```

**อ่านผลลัพธ์:**
- ค่า r ใกล้ +1 = สัมพันธ์เชิงบวกสูง
- ค่า r ใกล้ -1 = สัมพันธ์เชิงลบสูง
- ค่า r ใกล้ 0 = ไม่มีความสัมพันธ์
- ดู Sig. ว่า < 0.05 หรือไม่

---

### 7.7 Linear Regression (การถดถอยเชิงเส้น)

ใช้เมื่อต้องการพยากรณ์ตัวแปรตัวหนึ่งจากตัวแปรอื่น

**วิธีทำ:**
```
Analyze → Regression → Linear
→ Dependent: ตัวแปรที่ต้องการพยากรณ์ (Y)
→ Independent(s): ตัวแปรพยากรณ์ (X)
→ คลิก Statistics → ติ๊ก Estimates, Model fit, Descriptives
→ คลิก OK
```

**อ่านผลลัพธ์:**
- **R Square**: ค่าตัวแปร X อธิบายความแปรปรวนของ Y ได้กี่ %
- **B (Unstandardized Coefficient)**: ค่าสัมประสิทธิ์การถดถอย
- **Sig.**: ถ้า < 0.05 ตัวแปรนั้นมีผลต่อ Y อย่างมีนัยสำคัญ

---

### 7.8 Cronbach's Alpha (ความเชื่อมั่นของแบบสอบถาม)

ใช้ตรวจสอบว่าแบบสอบถามมีความสอดคล้องภายในหรือไม่

**วิธีทำ:**
```
Analyze → Scale → Reliability Analysis
→ ลากข้อคำถามทุกข้อในมิตินั้นเข้าช่อง Items
→ Model: Alpha (ค่า Default)
→ คลิก Statistics → ติ๊ก Item, Scale if item deleted
→ คลิก OK
```

**เกณฑ์:**
- Alpha ≥ 0.70 = ยอมรับได้
- Alpha ≥ 0.80 = ดี
- Alpha ≥ 0.90 = ดีมาก

## 8. การสร้างกราฟและแผนภูมิ

### Chart Builder (แนะนำ — ใช้ง่ายที่สุด)
```
Graphs → Chart Builder
→ เลือกประเภทกราฟจากแท็บ Gallery
→ ลากตัวแปรไปวางที่แกน X และ Y
→ คลิก OK
```

### ประเภทกราฟที่นิยม
| กราฟ | ใช้เมื่อ |
|---|---|
| Bar Chart | เปรียบเทียบกลุ่ม เช่น คะแนนจำแนกตามเพศ |
| Histogram | ดูการกระจายของข้อมูลต่อเนื่อง |
| Pie Chart | แสดงสัดส่วน เช่น % ของแต่ละกลุ่ม |
| Scatter Plot | ดูความสัมพันธ์ระหว่างตัวแปร 2 ตัว |
| Box Plot | ดูการกระจายและค่าผิดปกติ (Outlier) |
| Line Chart | แนวโน้มตามเวลา |

## 9. การบันทึกและส่งออกผลลัพธ์

### บันทึกไฟล์ข้อมูล
```
File → Save As → ตั้งชื่อไฟล์ → Save as type: SPSS Statistics (*.sav)
```

### ส่งออก Output เป็น Word/PDF/Excel
1. ใน Output Viewer คลิกขวาที่ตาราง หรือ
2. `File → Export` (ใน Output Viewer)
3. เลือกรูปแบบ: Word (.docx), PDF, Excel, PowerPoint
4. คลิก **OK**

### Copy ตารางไปวางใน Word
1. คลิกที่ตาราง Pivot Table ใน Output
2. คลิกขวา → **Copy**
3. เปิด Word → วาง (Ctrl+V)

### Use Case 1: วิจัยความพึงพอใจของนักศึกษา (Education Research)

**สถานการณ์**: อาจารย์ต้องการทราบว่านักศึกษามีความพึงพอใจต่อการเรียนการสอนออนไลน์แตกต่างกันตามชั้นปีหรือไม่

**ข้อมูลที่มี**: แบบสอบถาม Likert Scale 5 ระดับ จากนักศึกษา 200 คน

**การวิเคราะห์ที่ใช้ SPSS**:
1. **Cronbach's Alpha** — ตรวจสอบความเชื่อมั่นของแบบสอบถาม (ควรได้ ≥ 0.70)
2. **Descriptive Statistics** — ค่าเฉลี่ยและส่วนเบี่ยงเบนมาตรฐานของแต่ละข้อ
3. **One-Way ANOVA** — เปรียบเทียบความพึงพอใจเฉลี่ยระหว่างนักศึกษาปี 1, 2, 3, 4
4. **Post Hoc (Tukey)** — ดูว่าชั้นปีใดบ้างที่แตกต่างกัน

**ขั้นตอนย่อ**:
```
Step 1: นำเข้า Excel ที่รวบรวมข้อมูลแบบสอบถาม
Step 2: Compute Variable → คำนวณคะแนนรวมความพึงพอใจ
Step 3: Analyze → Scale → Reliability Analysis
Step 4: Analyze → Compare Means → One-Way ANOVA
Step 5: Export Output → Word → นำไปเขียนรายงานวิจัย
```

**ผลที่ใช้ประโยชน์ได้**: ทราบว่าชั้นปีใดมีความพึงพอใจต่ำที่สุด และควรปรับปรุงอะไร

### Use Case 2: วิเคราะห์พฤติกรรมผู้บริโภค (Marketing Research)

**สถานการณ์**: บริษัทต้องการรู้ว่า อายุ รายได้ และการศึกษา มีความสัมพันธ์กับยอดการใช้จ่ายต่อเดือน หรือไม่

**ข้อมูลที่มี**: ข้อมูลลูกค้า 500 ราย จากฐานข้อมูล CRM ที่ Export เป็น Excel

**การวิเคราะห์ที่ใช้ SPSS**:
1. **Pearson Correlation** — ดูความสัมพันธ์ระหว่าง อายุ/รายได้ กับยอดใช้จ่าย
2. **Multiple Regression** — พยากรณ์ยอดใช้จ่ายจากตัวแปรทั้ง 3 ตัวพร้อมกัน
3. **Crosstabs + Chi-square** — ดูว่าการซื้อสินค้าหมวดหมู่ A เกี่ยวข้องกับเพศหรือไม่

**ตัวอย่างผลลัพธ์ที่ได้**:
```
รายได้มีความสัมพันธ์เชิงบวกกับยอดใช้จ่าย (r = 0.62, p < 0.001)
โมเดลพยากรณ์ด้วย 3 ตัวแปรอธิบายความแปรปรวนได้ 45.3% (R² = 0.453)
→ ทีมการตลาดใช้ข้อมูลนี้แบ่งกลุ่มลูกค้าและวางแผนแคมเปญ
```

### Use Case 3: วิจัยทางการแพทย์ / สาธารณสุข (Healthcare Research)

**สถานการณ์**: โรงพยาบาลต้องการเปรียบเทียบประสิทธิผลของยา 2 ชนิด ในการลดความดันโลหิต

**ข้อมูลที่มี**: ผู้ป่วย 80 คน แบ่งเป็น 2 กลุ่ม วัดความดันก่อน-หลังรักษา

**การวิเคราะห์ที่ใช้ SPSS**:
1. **Paired Samples T-Test** — เปรียบเทียบความดันก่อน vs หลัง ภายในกลุ่มเดียวกัน
2. **Independent Samples T-Test** — เปรียบเทียบผลลัพธ์ระหว่างกลุ่มยา A กับยา B
3. **Descriptive Statistics** — ค่าเฉลี่ยความดันในแต่ละกลุ่ม
4. **Box Plot** — แสดงการกระจายของค่าความดันในแต่ละกลุ่ม

**ขั้นตอนเพิ่มเติม**:
```
Transform → Compute Variable
→ ตั้งชื่อ: blood_pressure_change
→ สูตร: bp_before - bp_after
→ ได้ตัวแปรใหม่ที่แสดงการเปลี่ยนแปลง
```

**ประโยชน์**: แพทย์และคณะกรรมการวิจัยใช้ผลลัพธ์ตัดสินใจเลือกโปรโตคอลการรักษา

### Use Case 4: ประเมินประสิทธิภาพพนักงาน (HR Analytics)

**สถานการณ์**: ฝ่าย HR ต้องการทราบว่าปัจจัยใดส่งผลต่อคะแนนการประเมินพนักงาน และอัตราการลาออกสัมพันธ์กับแผนกหรือไม่

**ข้อมูลที่มี**: ข้อมูลพนักงาน 300 คน (คะแนนประเมิน, อายุงาน, แผนก, เงินเดือน, อัตราการลาหยุด)

**การวิเคราะห์ที่ใช้ SPSS**:
1. **Correlation** — ความสัมพันธ์ระหว่างอายุงาน/เงินเดือน กับคะแนนประเมิน
2. **One-Way ANOVA** — เปรียบเทียบคะแนนเฉลี่ยระหว่างแผนก
3. **Logistic Regression** — พยากรณ์โอกาสที่พนักงานจะลาออก (0=ไม่ออก, 1=ออก)
4. **Crosstabs** — จำนวนพนักงานลาออกแยกตามแผนกและระดับตำแหน่ง

**ประโยชน์จริง**:
```
→ รู้ว่าแผนกไหนมีความเสี่ยงสูง (ควรเพิ่ม Retention Program)
→ รู้ว่าระดับเงินเดือนมีผลต่อการลาออกมากน้อยแค่ไหน
→ นำเสนอ Insight ให้ผู้บริหารด้วยกราฟและตารางที่ชัดเจน
```

### Use Case 5: วิเคราะห์ผลการทดสอบก่อน-หลังการฝึกอบรม (Training Evaluation)

**สถานการณ์**: บริษัทจัดอบรมพนักงาน 50 คน ต้องการดูว่าพนักงานมีความรู้เพิ่มขึ้นหลังฝึกอบรมจริงหรือไม่

**ข้อมูลที่มี**: คะแนนทดสอบก่อนอบรม (Pre-test) และหลังอบรม (Post-test) ของพนักงาน 50 คน

**การวิเคราะห์ที่ใช้ SPSS**:
1. **Paired Samples T-Test** — เปรียบเทียบ Pre-test กับ Post-test
2. **Descriptive Statistics** — ค่าเฉลี่ยของทั้ง 2 ช่วง
3. **Histogram** — ดูการกระจายคะแนนก่อนและหลัง
4. **Effect Size (Cohen's d)** — คำนวณขนาดผลได้ผ่าน Compute Variable

**ตัวอย่างการตั้งค่าข้อมูล**:
```
Variable View:
┌───────────┬─────────┬──────────────────────┐
│ Name      │ Type    │ Label                │
├───────────┼─────────┼──────────────────────┤
│ emp_id    │ Numeric │ รหัสพนักงาน           │
│ pretest   │ Numeric │ คะแนนก่อนอบรม        │
│ posttest  │ Numeric │ คะแนนหลังอบรม        │
│ dept      │ Numeric │ แผนก (1=Sales 2=IT)  │
└───────────┴─────────┴──────────────────────┘
```

**ประโยชน์**: HR นำผลไปรายงานผู้บริหารว่าการฝึกอบรมคุ้มค่าหรือไม่ และควรปรับปรุงหัวข้อใด

## 11. คำศัพท์ที่ควรรู้

| คำศัพท์ | ความหมาย |
|---|---|
| **Variable** | ตัวแปร — สิ่งที่วัดหรือสังเกต |
| **Case** | หนึ่งหน่วยการวิเคราะห์ (เช่น 1 คน, 1 บริษัท) |
| **Scale** | ข้อมูลตัวเลขต่อเนื่อง มีหน่วยวัด |
| **Nominal** | ข้อมูลกลุ่ม ไม่มีลำดับ เช่น เพศ, สี |
| **Ordinal** | ข้อมูลมีลำดับ แต่ห่างไม่เท่ากัน เช่น Likert Scale |
| **Mean** | ค่าเฉลี่ย |
| **SD (Std. Dev.)** | ส่วนเบี่ยงเบนมาตรฐาน |
| **p-value / Sig.** | ค่าความน่าจะเป็นสำหรับการทดสอบสมมติฐาน |
| **Null Hypothesis** | สมมติฐานหลัก (H0) ที่กล่าวว่าไม่มีความแตกต่าง |
| **Alpha (α)** | ระดับนัยสำคัญ — ปกติกำหนดที่ 0.05 |
| **Output Viewer** | หน้าต่างแสดงผลลัพธ์การวิเคราะห์ |
| **.sav** | นามสกุลไฟล์ข้อมูลของ SPSS |
| **.spv** | นามสกุลไฟล์ Output ของ SPSS |
| **Recoding** | การแปลงค่าตัวแปรใหม่ เช่น จากตัวเลขเป็นกลุ่ม |
| **Compute Variable** | การสร้างตัวแปรใหม่จากสูตรคำนวณ |
| **Missing Values** | ค่าขาดหาย / ข้อมูลที่ไม่ได้ตอบ |

## 12. ข้อผิดพลาดที่พบบ่อยและวิธีแก้ไข

### ปัญหา 1: "ตัวแปรไม่ปรากฏในหน้าต่าง Analyze"
**สาเหตุ**: ตั้งค่า Measure ผิด เช่น ตั้งเป็น Nominal แต่ต้องการใช้กับ Scale
**แก้ไข**: ไปที่ Variable View → แก้ไขค่า Measure ให้ถูกต้อง

### ปัญหา 2: "ผลลัพธ์ไม่ขึ้น หรือ Error Syntax"
**สาเหตุ**: ข้อมูลมีค่า Missing (ช่องว่าง) มากเกินไป
**แก้ไข**:
```
Analyze → Descriptive Statistics → Descriptives
→ ดูว่า N มีค่าน้อยกว่าจำนวนข้อมูลจริงหรือไม่
→ ถ้าใช่ → Data → Select Cases → ตรวจสอบข้อมูลขาดหาย
```

### ปัญหา 3: "นำเข้า Excel แล้วข้อมูลผิดพลาด"
**สาเหตุ**: มีแถวที่ว่าง หรือ Header ซ้อนกัน 2 แถวใน Excel
**แก้ไข**: แก้ไข Excel ให้แถวแรกสุดเป็น Header และข้อมูลเริ่มแถวที่ 2

### ปัญหา 4: "ค่า p ขึ้นว่า .000 หมายความว่าอะไร?"
**คำตอบ**: .000 หมายถึง p < 0.001 (น้อยกว่า 0.001) ไม่ใช่ค่าศูนย์จริงๆ
ยังถือว่ามีนัยสำคัญทางสถิติ (p < 0.05) อยู่แน่นอน

### ปัญหา 5: "ต้องการเปลี่ยนภาษาโปรแกรมเป็นอื่น"
```
Edit → Options → Language → เลือกภาษา → OK → รีสตาร์ทโปรแกรม
```
